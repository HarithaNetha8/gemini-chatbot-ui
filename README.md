
# Gemini Chatbot UI

A **GUI-based AI Chatbot** built using **Python Tkinter** and **Google Gemini API**. Interact with an AI in real-time through a clean and scrollable chat interface.

## 🚀 Features
- 🖥️ Interactive chat interface with Tkinter  
- ⚡ Real-time AI responses powered by Google Gemini API  
- 🎨 Simple, clean, and user-friendly UI  
- 📜 Scrollable chat window for full conversation history  

## 📦 Installation

1. **Clone the repository:**
```bash
git clone https://github.com/HarithaNetha8/gemini-chatbot-ui.git
cd gemini-chatbot-ui
````

2. **Create a virtual environment (optional but recommended):**

```bash
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
```

3. **Install dependencies:**

```bash
pip install -r requirements.txt
```

4. **Set up Google Gemini API credentials**
   Make sure to follow Google’s API documentation to obtain your API key and set it in your environment variables or config file.

## 🧪 How to Run

```bash
python main.py
```

* The chat window will open.
* Type your message and press Enter or click Send to get AI responses.

## 💡 How It Works

* Tkinter handles the GUI and user interactions
* User input is sent to the Google Gemini API
* API responses are displayed in the scrollable chat window

## ⚙️ Customization

* Update the UI design using Tkinter widgets and styling
* Add more features like voice input, emojis, or themes
* Integrate logging to save chat history
